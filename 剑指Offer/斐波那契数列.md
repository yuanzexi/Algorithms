题目描述
大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项。
n<=39

Solution:(java)
```java
public class Solution {
    public int Fibonacci(int n) {
        /*
        Ideas: Dynamic Programming: f(n) = f(n-1) + f(n-2)
        1. Recursive method is space-consuming.
        2. Calculate from bottom to top. (Avoiding repeat calculation)
        
        */
        if (n < 2){
            return n;
        }
        int nMinusOne = 1;
        int nMinusTwo = 0;
        int result = 0;
        for (int i = 2; i <= n; i++){
            result = nMinusOne + nMinusTwo;
            nMinusTwo = nMinusOne;
            nMinusOne = result;
        }
        return result;
    }
}
```
# 矩形覆盖问题
题目描述
我们可以用2*1的小矩形横着或者竖着去覆盖更大的矩形。请问用n个2*1的小矩形无重叠地覆盖一个2*n的大矩形，总共有多少种方法？
Solution:(java)
```java
public class Solution {
    public int RectCover(int target) {
        /*
        Ideas: Actually, it is the same as Fibonacci problem.
        Dynamic Programming: f(n) = f(n-1) + f(n-2)
        1. Recursive method is space-consuming.
        2. Calculate from bottom to top. (Avoiding repeat calculation)
        
        */
        if (target < 3){
            return target;
        }
        int nMinusOne = 2;
        int nMinusTwo = 1;
        int result = 0;
        for (int i = 3; i <= target; i++){
            result = nMinusOne + nMinusTwo;
            nMinusTwo = nMinusOne;
            nMinusOne = result;
        }
        return result;
    }
}
```
